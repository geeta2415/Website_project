🌸 Éclat d'Aroma — Perfume E-Commerce Website Éclat d'Aroma is a multi-page perfume e-commerce website built with HTML and CSS as a 1st semester course project at SZABIST, Karachi. The website represents a luxury perfume brand with an elegant design that reflects timeless sophistication. The site has five files: index.html is the homepage featuring a hero section, limited edition seasonal offers, customer testimonials, and a newsletter signup. shop.html is the shop page displaying four signature perfumes — Noir Éclat, Fleur Essence, Citrus Allure, and Velvet Woods — each with an image, description, and price. about.html covers the brand story, mission, and sustainability values. guide.html is a fragrance guide that breaks down scent families including Fresh & Citrusy, Floral & Romantic, Woody & Earthy, and Spicy & Exotic. All styling is handled through a single styles.css file. Features include a responsive layout, a consistent navigation bar across all pages with a Shop Now button, a language selector (EN/FR/ES), product cards with pricing in PKR, a fragrance guide with images, customer testimonial cards, a newsletter subscription form, and a footer with contact information, quick links, and social media icons. This project was built using HTML5 and CSS3 only, with no frameworks or JavaScript libraries. To run it, simply download the repository and open index.html in any web browser.
